# 危大工程方案智能审核系统产品手册

## 一、核心功能

1. 多专业智能识别：自动识别模板支撑工程、起重吊装及起重机械安装拆卸工程、深基坑工程、脚手架工程、拆除、爆破工程、暗挖工程、建筑幕墙安装工程、人工挖孔桩工程、钢结构安装工程、水下作业工程、装配式建筑混凝土预制构件安装工程、采用新技术、新工艺、新材料、新设备工程、有限空间作业。
2. 精准规范引用：每个专业配置独立审核锚点库，按专业和构造类型调度，减少跨专业误引规范。
3. 三层审核架构：先审致命缺陷，再审专业技术问题，最后审管理与完整性问题。
4. 结构化报告输出：按问题性质归类，输出规范依据、问题说明、整改建议和结论。

## 三、技术优势

1. 解决 AI 幻觉：审核依据来自结构化条款锚点库，报告引用可追溯到规范名称、条款编号和审核要点。
2. 多专业调度：先识别专业和构造类型，再加载对应专业弹药库，避免用模板条款审脚手架、用脚手架条款审模板支撑。
3. 专家级审核：围绕危大工程高风险点设计检查项，优先发现会导致失稳、倾覆、坠落、超载、验算缺失的问题。
4. 快速整改闭环：报告直接指出问题、依据和整改方向，便于施工单位、监理单位和专家论证人员复核。

## 四、十三专业审核清单

| 专业 | 致命缺陷优先项 | 技术问题优先项 | 管理问题优先项 |
|------|----------------|----------------|----------------|
| 模板支撑工程 | 支撑体系参数错误、超规模未论证、监测缺失 | 立杆规格、步距、可调托座外露、剪刀撑布置 | 方案审批、交底、验收、材料进场证明 |
| 起重吊装工程 | 吊装能力不足、双机抬吊未计算、试吊缺失 | 吊索安全系数、地基承载、支腿布置、风速控制 | 起重设备证照、人员持证、警戒区、应急措施 |
| 深基坑工程 | 稳定性验算缺失、周边保护缺失、降水失控 | 支护参数、监测点、报警值、开挖工况 | 专家论证、第三方监测、巡视检查、信息反馈 |
| 脚手架工程 | 卸料平台不独立、钢丝绳不参与平台受力、自由高度超限 | 连墙件刚性、绳夹数量、悬挑梁锚固、吊篮安全绳 | 限载牌、验收记录、使用维护、作业人员管理 |
| 拆除、爆破工程 | 立体交叉拆除、先拆承重结构、爆破安全距离缺失、盲炮处置错误 | 拆除顺序、机械作业高度、人机隔离、防护覆盖、飞散物距离 | 爆破公告、人员资格、三岗分离、安全技术交底 |
| 暗挖工程 | 开仓压力无计算试验、仓内作业未锁定、冻结壁温度未达标、监测预警失效 | 土仓/泥水压力、同步注浆、初期支护、出土量、通风供风 | 风险分级、超前地质预报、监测报送、应急处置 |
| 建筑幕墙安装工程 | 结构胶无相容性报告、锚固失效、玻璃或板块坠落风险 | 横梁立柱厚度、挂件厚度、孔边距、层间封堵 | 材料证明、检测报告、安装验收、隐蔽验收 |
| 人工挖孔桩工程 | 禁用地层仍采用挖孔桩、孔内缺氧中毒、边抽水边作业 | 护壁厚度强度、孔深桩径、送风检测、串筒灌注 | 安全交底、监护记录、气体检测记录、桩身检测 |
| 钢结构安装工程 | 无焊接工艺评定、双机抬吊未计算、临时支撑提前拆除 | 高强螺栓终拧、摩擦面抗滑移、探伤、整体偏差 | 焊工资质、检测报告、吊装指挥、验收记录 |

| 装配式建筑混凝土预制构件安装工程 | 吊具无验算、临时支撑不足、灌浆不饱满 | 钢筋定位、模拟接头、吊索夹角、楼梯搁置长度 | 构件保护、灌浆记录、试件强度、安装验收 |
| 采用新技术、新工艺、新材料、新设备工程 | 强标偏离未核准、限禁技术违规采用、无专题论证 | 技术特性、试验验证、专家论证、推广应用措施 | 申请资料、论证意见、审查记录、风险控制 |

## 五、产品边界

1. 当前系统已经跑通十三个专业：模板支撑、起重吊装、深基坑、脚手架、拆除爆破、暗挖、建筑幕墙、人工挖孔桩、钢结构安装、水下作业、装配式预制构件安装、四新工程、有限空间作业。
2. 脚手架本地依据库当前包含 JGJ 130-2011，尚未纳入 JGJ 130-2024 原文；涉及新版条款时需先补充本地规范依据。
3. 第 7~13 专业当前已具备核心条款清单，后续可继续扩充幕墙防火封堵专项、挖孔桩地方禁限用、钢结构复杂吊装、水下焊接切割、装配式节点专项、四新技术评估、有限空间行业细则等更细分条款。
4. 混合方案能力已支持“主专业 + 外架/平台章节”的条款带入；跨主专业并行调度（例如深基坑 + 高大模板、钢结构 + 起重吊装）需要单独验证和增强。
